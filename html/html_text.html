文本缩进

把 Web 页面上的段落的第一行缩进，这是一种最常用的文本格式化效果。 CSS 提供了 text-indent 属性，该属性可以方便地实现文本缩进。 通过使用 text-indent 属性，所有元素的第一行都可以缩进一个给定的长度，甚至该长度可以是负值。 这个属性最常见的用途是将段落的首行缩进，下面的规则会使所有段落的首行缩进 5 em：
